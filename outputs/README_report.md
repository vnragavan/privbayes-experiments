# Generated figures and tables (RQ1–RQ5)

Figures and tables are produced by the pipeline; **report.pdf / report.tex are optional** and are not part of the repo. You can generate figures and tables only, or optionally build a local report.

## Figures and tables (main outputs)

- **`outputs/figures/`** — PDFs: compliance heatmap, utility curves, survival curves, privacy risk, performance.
- **`outputs/tables/`** — TeX: compliance audit, utility/survival, privacy performance.

These are generated by `analysis.generate_all` and are **not tracked** (reproducible locally).

## Optional: build report PDF locally

If you have `report.tex` (not in repo), compile from **outputs**:
```bash
cd outputs && pdflatex report.tex && pdflatex report.tex
```

## Rerunning experiments

From the **repository root**:

1. **Sweep** (run all implementations over ε and seeds; writes CSVs):
   ```bash
   python run_sweep.py --schema schemas/lung_schema.json --data data/lung_clean.csv
   ```
   Output: `results/eps_sweep/lung_clean/*.csv`

2. **Compute metrics** (build result JSONs from CSVs):
   ```bash
   python compute_sweep_metrics.py \
     --schema schemas/lung_schema.json \
     --real-data data/lung_clean.csv \
     --sweep-dir results/eps_sweep/lung_clean \
     --out-dir results/eps_sweep/lung_clean
   ```
   Output: `results/eps_sweep/lung_clean/results_eps*_seed*.json`

3. **Generate figures and tables**:
   ```bash
   python -m analysis.generate_all \
     --results-dir results/eps_sweep/lung_clean \
     --figures-dir outputs/figures \
     --tables-dir outputs/tables
   ```
   Output: `outputs/figures/fig1_*.pdf` … `fig6_*.pdf`, `outputs/tables/tab1_*.tex` … `tab3_*.tex`

4. **(Optional)** Recompile report PDF from `outputs/report.tex` if you have it.
